## Loops

- for
```javascript
for(let i = 0; i < array.length; i++){
    array[i]
}
```

- for..in
```javascript
for (const i in iterable) {
  console.log(i);
}
```

- for..of
```javascript
for (const element of array1) {
  console.log(element);
}
```
